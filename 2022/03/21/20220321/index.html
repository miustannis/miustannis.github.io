插入图片

之前打算在文章中插入图片，配置的过程中遇到了一些问题。想在博客中引用外站图片问题不大，但hexo默认只有这一个方法，对有本地化需求的人群来说不够友善，具体改进方案是安装一个满足本地图片引用的插件，在国内某知名专业开发者社区里提供的90%的方案为以下命令：
npm install hexo-asset-image –save
然而这一版本存在极大的bug，简单来说就是根本无法显示图片，该论坛上相互借鉴蔚然成风的壮景对我这个刚入门的小白真是非常险恶。

正确有效的版本安装命令如下：
npm install https://github.com/CodeFalling/hexo-asset-image –save

修改_config.yml文件，将post_asset_folder:值改为true。
